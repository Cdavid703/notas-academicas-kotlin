# Notas Académicas Kotlin

Aplicación Android desarrollada en Kotlin para calcular la nota definitiva de un estudiante.

## Funcionalidades
- Ingreso del nombre del estudiante
- Ingreso del nombre de la materia
- Registro de notas de conocimiento, desempeño y producto
- Cálculo de nota definitiva
- Mensaje emergente indicando si ganó o perdió
- Visualización del resultado en pantalla

## Tecnologías
- Kotlin
- Android Studio
- XML
